    • This is my personal website. I am building the website to mimic the New York Times' site. I particularly admire their use of whitespace to creating an aesthetically pleasing user experience. My future goals for this website are to create a sticky nav bar up top on pages other than this one, i.e., index.html, to layout the pages using columns, to incorporate more photos and possibly some videos.
