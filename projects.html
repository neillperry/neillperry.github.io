Movie Battle - React JS

Code
This is a code along project that served as an introduction to ReactJS. The code follows the introductory course once taught by Tyler McGinnis. The web application prompts users to select two movies to battle against one another. Once selected, the applicaiton queries the OMDB API for the respective movies' rankings. The movie with the higher ranking wins.
